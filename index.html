Wi-Fi Custom DAQ & Firing Unit

Engineered custom high-speed data acquisition boards for solid propellant static fire setups. Telemetry is streamed wirelessly at 10 Hz via an ESP8266 serial bridge, utilizing transistor-driven MOSFET relays to enforce strict hardware safe-arm steps.

CHOTU Self-Balancing Robot

A two-wheeled inverted pendulum robot built on the Shrike Lite RP2040 dev board. Implemented cascaded PID controller loops in MicroPython, fusing IMU registers to compute pitch angles. Upgraded with real-time Wi-Fi telemetry and remote calibration loops.

MicroPython PID Loop IMU Fusion Tele-Control

nRF52 Health Wearable POC

Developed firmware blocks on an nRF52 SoC utilizing the Zephyr RTOS framework. Configured hardware interrupts to pull vital biometric sensor data and broadcast frames using Bluetooth Low Energy (BLE) peripheral notification channels.

nRF52 series Zephyr RTOS Bluetooth LE Proof of Concept

STM32 Custom Bare-Metal Libraries

Wrote a robust custom hardware abstraction library from scratch in bare C for STM32 cores. Directly configured system clock trees (PLL/HSE), GPIO registers, hardware watchdog timers, and hardware timers without relying on vendor HAL tools.

Bare-metal C Clock Trees GPIO Drivers Timer registers

STM32 USB-HID Device Emulator
